# Aesthetic Shopping App

A Flutter shopping app for beauty products with bilingual support.

## Features
- ✅ User registration and login
- ✅ Product catalog browsing  
- ✅ Shopping cart functionality
- ✅ English/Arabic language support
- ✅ Hot offers section
- ✅ Responsive design
